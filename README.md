# Admin Web Panel

TypeScript-based admin panel for managing users, verifications, and system statistics.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

```
admin-web/
├── src/
│   ├── pages/          # Page components
│   ├── router.ts       # Routing logic
│   ├── styles/         # CSS files
│   └── main.ts         # Entry point
├── index.html
├── package.json
├── tsconfig.json
└── vite.config.ts
```
